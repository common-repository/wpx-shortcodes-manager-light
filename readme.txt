=== NOTE ===
**This plugin is DEPRECATED**, and its features have been integrated into [Scotty](https://wordpress.org/plugins/scotty/).
**Download** [Scotty](https://wordpress.org/plugins/scotty/) **to continue using these features**.

Shortcodes Manager provides you with the cleanest and simplest tool to manage WordPress shortcodes you've ever seen.

= Features =

* Write your shortcode with the shortcode editor
* Display all registered shortcodes
* Disable/enable all registered shortcodes
* Enable a Shortcode Analysis, checking your post content while editing

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the entire content of plugin archive to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress (deactivate and reactivate if you're upgrading).
3. Done. Enjoy.
